# OpenChoreo Case Studies

A curated collection of example projects and real-world case studies built on top of [OpenChoreo](https://github.com/openchoreo/openchoreo) — designed to help developers explore, learn, and adopt OpenChoreo in their own workflows.

## 🌐 What is OpenChoreo?

[OpenChoreo](https://github.com/openchoreo/openchoreo) is an open-source internal developer platform that enables teams to build, deploy, and manage cloud-native applications with ease. It provides a developer-friendly abstraction over Kubernetes and other cloud infrastructure.

## 📚 About This Repository

This repository contains a growing library of:

- **Example Projects** – Minimal, focused examples that demonstrate specific OpenChoreo features
- **Case Studies** – End-to-end real-world scenarios built using OpenChoreo

Whether you are just getting started or looking to implement advanced patterns, these resources will help you understand how to make the most of OpenChoreo.

## 🗂️ Repository Structure

```
openchoreo-case-studies/
├── examples/           # Small, focused feature examples
├── case-studies/       # End-to-end real-world case studies
└── docs/               # Supporting documentation and guides
```

## 🚀 Getting Started

### Prerequisites

- A running OpenChoreo environment — see the [OpenChoreo documentation](https://github.com/openchoreo/openchoreo) to get started
- Basic familiarity with Kubernetes and cloud-native concepts

### Using a Case Study or Example

1. Browse the [`examples/`](./examples/) or [`case-studies/`](./case-studies/) directories
2. Navigate into a project folder
3. Follow the `README.md` inside that folder for setup and usage instructions
